虚拟机内存
===

### 运行时数据区
根据《Java虚拟机规范（Java SE 7版）》的规定，Java虚拟机所管理的内存将会包括以下几个运行时数据区域:
* 方法区
* 虚拟机栈
* 本地方法栈
* 堆
* 程序计数器

---
##### 线程作用范围
1. 线程共享  
方法区  
堆
2. 线程独享  
虚拟机栈  
本地方法栈  
程序计数器  

---

##### 内存溢出异常
1. StackOverflowError异常  
虚拟机栈  
本地方法栈  
2. OutOfMemoryError异常  
虚拟机栈  
本地方法栈  
堆  
方法区  
运行时常量池(方法区的一部分)  
直接内存  

> 程序计数器是唯一没有规定内存溢出异常的数据区.
